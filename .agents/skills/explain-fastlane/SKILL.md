# Explain Fastlane

Activate only from an explicit owner request.

Begin exactly:

`Learning mode — explanation only; no project decision changed.`

Explain the requested topic using current project IDs, evidence, alternatives,
and tradeoffs when relevant. Before Gate A, label every named AWS stack:

`Illustrative architecture candidate — not selected or approved.`

Do not edit files, alter requirements or design, accept assumptions, approve a
gate, create tasks, authorize external work, or invoke AWS account APIs. End by
stating that project state did not change and restore the exact pending owner
action from the coordinator. `LEARN-10` is only a compatibility alias for this
skill; it is not an automatic lifecycle route.
